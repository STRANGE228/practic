> [!TIP]
> Установка библиотек
```
pip install -r requirements.txt
```
> [!TIP]
> Запуск приложения
```
uvicorn app.main:app --reload
```
> [!NOTE]
> Использованные технологии
- Backend:	Python 3.11+, FastAPI
- Database:	SQLite (SQLAlchemy ORM)
- Frontend:	HTML, Jinja2, CSS, JavaScript
- Auth:	JWT, bcrypt
